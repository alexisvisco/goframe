# Tasks

Tasks are simple commands executed from the CLI. They are handy for one‑off scripts or data migrations that don’t belong in a workflow.

Generate a new task with:

```bash
bin/goframe generate task <task name> --description "what the task does"
```

This creates `internal/task/task_<task name>.go` and registers it with the CLI so you can run it directly.

A generated task looks like:

```go filename="task_cleanup.go"
type CleanupTask struct {
    // declare dependencies with fx.In
}

func (t CleanupTask) Run(ctx context.Context, opts *CleanupTaskOptions) error {
    // implement the task logic here
    return nil
}
```

Run the task using:

```bash
bin/goframe task <task name>
```

See [Task Runner](./task-runner.mdx) for how the CLI integrates with FX.
For details on customizing commands read the [CLI documentation](/docs/cli).
